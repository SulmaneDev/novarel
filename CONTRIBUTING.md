# Contributing to Novarel

We welcome contributions to Novarel! This guide will help you get started.

---

## 🧱 Project Setup

1. Clone the repo:
   ```bash
   git clone https://github.com/yourname/novarel.git
   cd novarel
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Run dev build or tests:
   ```bash
   pnpm build
   pnpm test
   ```

---

## 📐 Coding Guidelines

- Use **TypeScript** with strict types.
- Run `pnpm lint` before committing.
- Use [Conventional Commits](https://www.conventionalcommits.org/):
  - `feat:` for new features
  - `fix:` for bug fixes
  - `chore:` for setup/config changes
  - `docs:` for documentation changes
  - `refactor:` for internal code changes

---

## 💬 Contribute via

- Bug reports → GitHub issues
- Feature requests → GitHub discussions
- Pull requests → Fork + branch + PR
