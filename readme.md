> **Novarel** — A Laravel-inspired, batteries-included Node.js framework built with TypeScript.

Novarel offers a clean, opinionated architecture and includes essential features out of the box: routing, controllers, dependency injection, request validation, middleware, configuration, testing, and more. Designed for developers who want structure without sacrificing flexibility.

---

## ✨ Features

- ✅ Expressive routing system
- ✅ Dependency injection container
- ✅ Request validation with decorators
- ✅ Middleware support (global and route-specific)
- ✅ Modular service providers
- ✅ Built-in support for testing (Jest)
- ✅ Preconfigured ESLint, Prettier, Husky, and lint-staged
- ✅ TypeScript-first with strict typing
- ✅ Git and NPM-ready for scalable development
