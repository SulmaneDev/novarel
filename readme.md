# Novarel

> ⚠️ Novarel is currently under active development. It is **not production-ready** yet.

A modern, expressive Node.js framework inspired by Laravel.  
Novarel aims to bring elegant routing, middleware, validation, and request/response handling to Node.js — with a Laravel-like developer experience and full TypeScript support.

---

## ✨ Features (Planned)

- ⚡ Fluent routing system
- ✅ Laravel-style validation using rule strings
- 🔁 Middleware and lifecycle hooks
- 🎯 Request/Response abstraction (like Laravel's `Request` & `Response`)
- ⚙️ Extensible architecture
- 📦 ESM + CJS support
- 💡 TypeScript-first with typed inputs/outputs
